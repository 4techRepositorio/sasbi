Ao criar endpoint:
1. Criar schema de request/response.
2. Criar router.
3. Criar service.
4. Criar repository se necessário.
5. Adicionar tenant isolation.
6. Adicionar tratamento de erro.
7. Adicionar teste mínimo.
8. Atualizar docs se impactar contrato.
