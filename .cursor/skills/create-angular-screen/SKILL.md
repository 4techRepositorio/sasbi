Ao criar tela:
1. Criar página/container.
2. Criar componentes auxiliares.
3. Criar serviço de integração com API.
4. Tratar loading, erro, vazio e sucesso.
5. Garantir responsividade.
6. Seguir padrão visual corporativo e limpo.
